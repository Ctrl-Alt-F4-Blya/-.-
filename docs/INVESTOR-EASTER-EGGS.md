# Небольшие инвесторские заметки

На стартовых экранах страниц есть компактные интерактивные карточки «Инвесторская заметка».

- Нажатие на заметку переключает короткий практический совет.
- Подсказки не являются инвестиционной рекомендацией.
- Тексты находятся в HTML-атрибуте `data-tips` у каждого блока `[data-investor-tip]`.
