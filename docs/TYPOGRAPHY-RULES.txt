НОВЫЙ ДОМ — ТИПОГРАФИКА

Основной шрифт сайта: FindSans Pro.

Правила:
1. Заголовки не дробить ручными <br> внутри слов.
2. Для нового длинного заголовка: один смысл на 1–3 строки, без слов-одиночек в последней строке.
3. Основной текст: 15–16 px, line-height 1.6–1.7.
4. Вторичный текст на светлом фоне использовать через глубокий теплый коричневый с прозрачностью, не светло-серым #C2A792.
5. На темном фоне использовать белый и светло-серый #F1D8AF с высокой непрозрачностью.
6. Перед добавлением нового блока проверить ширину 1440 px, 1280 px, 1024 px и 390 px.
