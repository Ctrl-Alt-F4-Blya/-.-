# Новый дом — премиальная версия сайта

## Что внутри

- 10 связанных HTML-страниц: главная, о компании, инвесторам, управление, кейсы, отдельный кейс, калькулятор, FAQ, контакты и политика конфиденциальности;
- адаптивная верстка для телефонов, планшетов и десктопов;
- локальные изображения, CSS-анимации, мобильное меню, FAQ-аккордеон и калькулятор;
- Express backend для заявок;
- отправка заявок в Telegram-бот;
- кнопки подтверждения времени / переноса созвона в Telegram;
- отправка нового времени клиенту по SMTP email.

## Запуск

1. Установите Node.js 18 или новее.
2. В папке проекта выполните:

```bash
npm install
cp .env.example .env
```

3. Заполните `.env`: минимум `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`. Для писем при переносе времени также заполните SMTP-поля.
4. Запустите проект:

```bash
npm start
```

Сайт откроется по адресу `http://localhost:3000`.

## Важно перед публикацией

- Замените демонстрационные телефон, email и юридическую политику конфиденциальности на реальные данные компании.
- В калькуляторе все цифры являются ориентировочными.
- На хостинге задайте переменные из `.env` через панель окружения и не загружайте сам `.env` в репозиторий.
